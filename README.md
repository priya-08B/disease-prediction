Multi-Disease Prediction System (Diabetes, Heart, Kidney)

A Streamlit-based AI Health Prediction System that predicts Diabetes, Heart Disease, and Kidney Disease using Machine Learning models.
Includes OCR-based input extraction and full multilingual support.

Features

1. Disease Predictions

Diabetes Prediction

Heart Disease Prediction

Kidney Disease Prediction

2. OCR-Based Input Extraction

Upload image/PDF of medical reports

Extracts values automatically using Tesseract OCR

3. Full Multilingual Support

4. User-Friendly Streamlit UI

Clean and responsive

Easy navigation between disease sections

5. Machine Learning Models

Each disease prediction uses a trained ML model (Logistic Regression / SVM / Random Forest / etc.).

Tech Stack

| Category                | Technologies                |
| ----------------------- | --------------------------- |
| **Frontend**            | Streamlit                   |
| **Backend**             | Python                      |
| **OCR Engine**          | Tesseract.js / pytesseract  |
| **ML Libraries**        | NumPy, Pandas, Scikit-learn |
| **Languages Supported** | EN, HI, TA, ES, JA, TE      |
